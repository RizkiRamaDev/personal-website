Personal Website

Website ini saya bangun sebagai media personal branding sekaligus sarana latihan pengembangan web. Proyek ini menjadi titik awal saya dalam membangun fondasi menuju Full Stack Developer.

Dalam proses pembuatannya, saya fokus pada struktur HTML yang rapi, penggunaan CSS dasar untuk layout dan responsivitas, serta JavaScript untuk interaksi sederhana seperti dark mode.

Tech Stack

  • HTML (Semantic)
  • CSS (Flexbox, Responsive)
  • JavaScript (DOM & Event)
  • Git & GitHub Pages

Apa yang Saya Pelajari

  • Menyusun struktur website yang siap dikembangkan
  • Menggunakan GitHub sebagai version control
  • Membangun fitur interaktif sederhana
  • Mengelola proyek secara bertahap

Proses & Tantangan

Dalam pengembangan website ini, tantangan utama saya adalah membangun struktur yang rapi sejak awal, meskipun kemampuan teknis saya masih berada di tahap fundamental.

Saya memilih untuk fokus pada dasar-dasar seperti semantic HTML, pemisahan struktur dan styling, serta penggunaan GitHub sebagai version control, agar website ini mudah dikembangkan di tahap berikutnya.

Selain itu, keterbatasan perangkat membuat saya harus beradaptasi dengan workflow berbasis browser. Dari situ saya belajar bahwa keterbatasan teknis bukan penghalang untuk tetap membangun dan mendokumentasikan proses belajar secara konsisten.
