Latar Belakang

Saya memulai karier sebagai Warehouse Admin di perusahaan produksi makanan. Dalam peran ini, saya terbiasa bekerja dengan data operasional, laporan stok, serta analisis penjualan menggunakan Microsoft Excel.

Pengalaman tersebut membentuk pola pikir saya yang terstruktur, detail, dan berorientasi pada efisiensi proses.

Peralihan ke Dunia Programming

Ketertarikan saya pada programming bukanlah hal baru. Saya merupakan lulusan Sistem Informasi, namun perjalanan karier membawa saya lebih dulu ke dunia operasional.

Saat ini saya memutuskan untuk kembali menekuni bidang teknologi secara serius, dimulai dari pengembangan web dan secara bertahap menuju full stack development.

Tujuan Karier

Tujuan saya adalah berkarier sebagai developer profesional, baik di bidang web, desktop, maupun aplikasi mobile, dengan lingkungan kerja yang mendorong pembelajaran berkelanjutan dan fleksibilitas kerja.
